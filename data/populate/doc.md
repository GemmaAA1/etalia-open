Journal Format
===============

```json
{
    'title':        (string),
    'short_title':  (string),   # Iso abbreviated title when available
    'id_issn':      (string),   # Print identifier 
    'id_eissn':     (string),   # Online/Electronic identifier
    'id_arx':       (string),   # Arxiv identifier
    'id_oth':       (string),   # Other identifier
    'period':       (string),   # publishing period recurrence
    'language':     (string),   # language of the journal
    'url':          (string),   # url of the journal page
    'publisher':    (string),   # publisher name
    'scope':        (string)    # 
}
```
